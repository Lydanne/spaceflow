## 系统概述

Teax 是 **Gitea 的功能扩展平台**，通过 Web UI 为 Gitea 提供增强能力，并通过 **飞书** 实现控制指令和消息交互。

### 核心架构

```text
┌─────────────────────────────────────────────────────────┐
│                        Teax                             │
│  ┌─────────────────┐           ┌─────────────────────┐  │
│  │     Gitea       │◀─────────▶│       飞书          │  │
│  │   功能扩展       │           │   控制 & 消息交互   │  │
│  └─────────────────┘           └─────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

- **Gitea 侧**：扩展 CI/CD、Pages 托管、小程序管理、Agent 运行时等功能
- **飞书侧**：消息通知、机器人指令、审批流程、状态查询

### 核心功能

- **项目管理** - 关联 Gitea 仓库，统一管理项目
- **CI/CD 集成** - Gitea Actions 集成，Workflow 管理
- **Agent 系统** - AI Agent 执行，open-code 集成
- **工作区** - 容器化开发环境，Web IDE
- **飞书集成** - 消息通知、机器人指令、审批流程
- **权限管理** - 权限组、团队权限、访问控制

详细信息请查看 **[架构概览](./overview.md)**。
